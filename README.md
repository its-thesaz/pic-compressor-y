## 🇬🇧 English Documentation

**PicCompressor Y** is an advanced, privacy-first, client-side web application designed for lightning-fast image compression, resizing, and manipulation. Featuring a fluid fluid-glassmorphic interface, it processes everything directly within the user's browser, ensuring absolute security with zero server overhead.

### 🔥 Key Features
- **⚡ Local Processing Engine:** Utilizes the high-performance Canvas API; your private images never touch any external server.
- **🖼️ Wide Format Support:** Import and optimize multiple formats (JPEG, PNG, WebP, BMP, GIF, TIFF, AVIF, HEIC, SVG, ICO).
- **🎛️ Precision Controls:** Real-time optimization sliders for Output Quality, Scale/Dimensions (with aspect ratio lock), and Rotation.
- **🎨 Visual Matrix Adjustments:** Live sliders for Brightness, Contrast, Saturation, and Gaussian Blur effects.
- **🎯 Native Edge Sharpening:** Built-in low-level kernel convolution filter for custom edge-sharpening.
- **🌐 Dynamic Localization:** Fully integrated English/Persian interface flipping natively with active RTL/LTR adjustments.
- **🌓 Persistence Themes:** Seamless Dark Mode and Light Mode with strict design tokens.

### 🛠️ Technology Stack
| Layer | Technologies Used |
| :--- | :--- |
| **Frontend UI** | HTML5, Advanced CSS3, Fluid Glassmorphic Design Variables |
| **Core Engine** | Pure Vanilla JavaScript (ES6+), Client-Side Canvas API |
| **Typography** | Vazirmatn Font Face Integration |

## 🇮🇷 توضیحات و ویژگی‌های فارسی

**PicCompressor Y** یک وب‌اپلیکیشن فوق‌پیشرفته، مدرن و حریم‌خصوصی‌محور برای بهینه‌سازی، کاهش حجم، تغییر ابعاد و ویرایش تصاویر است. این استودیو با بهره‌گیری از یک طراحی شیک شیشه‌ای (Glassmorphic) و پردازش کاملاً محلی، بدون نیاز به سرور امنیت کامل تصاویر شما را تضمین می‌کند.

### 🔥 ویژگی‌های کلیدی
- **⚡ هسته پردازش محلی:** پردازش مستقیم روی سخت‌افزار مرورگر با Canvas API بدون آپلود فایل‌ها در اینترنت.
- **🖼️ پشتیبانی جامع از فرمت‌ها:** قابلیت بهینه‌سازی و تبدیل فرمت‌های رایج و مدرن از جمله AVIF، WebP، JPEG، PNG و غیره.
- **🎛️ ابزارهای ابعادی دقیق:** اسلایدر کنترل کیفیت خروجی، ابزار تغییر رزولوشن مجهز به قفل نسبت تصویر و چرخش زاویه‌ای تصویر.
- **🎨 فیلترهای استودیویی زنده:** کنترل آنی پارامترهای روشنایی (Brightness)، کنتراست، اشباع رنگ و میزان تاری (Blur).
- **🎯 تیزسازی اختصاصی لبه‌ها:** مجهز به الگوریتم هوشمند بومی نیتو (Sharpen Effect) برای بازیابی جزئیات پس از فشرده‌سازی.
- **🌐 ناوبری دو زبانه هوشمند:** جابه‌جایی آنی و کاملاً بومی بین زبان‌های فارسی و انگلیسی همراه با تغییر جهت خودکار چیدمان صفحه (RTL/LTR).
- **🌓 مدیریت پوسته هوشمند:** تم‌های تاریک و روشن چشم‌نواز متناسب با نور محیط.

## 🚀 Deployment & Local Usage / راهنمای اجرا

### English
Since this project is built entirely on native client-side web technologies, it requires **no installation** or package management.
1. Clone the repository: `git clone https://github.com/its-thesaz/pic-compressor-y.git`
2. Open `index.html` directly in any modern web browser.

### فارسی
به دلیل استفاده از تکنولوژی‌های بومی کلاینت، این برنامه نیازی به نصب هیچ پیش‌نیازی ندارد.
۱. مخزن را کلون کنید یا فایل‌ها را دانلود کنید.
۲. فایل `index.html` را در مرورگر خود باز کنید تا برنامه فوراً اجرا شود.
